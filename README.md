# Terraria-Mod

## Description
This mod adds new elements and mechanics to Terraria, improving the gameplay.

## Features
- New bosses: Ice Dragon and Noon Beast.
- Unique weapons: Ice Sword, Ghost Bow.
- New mechanics: Spell creation with runes.
